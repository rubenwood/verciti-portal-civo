# verciti-portal-civo
This is the same as the verciti-portal-va repo, but with changes made to deploy on civo (Kubernetes cluster)

## Getting Started

You may need to run:
```bash
npm install
# or
npm install --legacy-peer-deps
# or
npm install --force
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
